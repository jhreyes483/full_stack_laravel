 ## Tecnologias
 - Laravel Framework 8.83.27
 - Angular v17

### 1. Entorno de desarrollo optimo
- npm -v 10.2.4
- node -v 20.9.0
- Composer  Dependency Manager for PHP - version 2.6.5
- php 7.4



### 2. Instalar dependencias
- npm i
- composer install
- npm update


### 3. Iniciar proyecto en mode desarrollo
- npm run dev

### 4. Nombre inicial DB
- api_rest_laravel
- MySQL

### 5. Host
- http://localhost:4200/
